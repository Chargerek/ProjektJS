# Social App Frontend

Frontend aplikacji społecznościowej zbudowany w React i Vite.

## Technologie
- React 19
- Vite
- JavaScript (ES6+)

## Instalacja

```bash
npm install
```

## Uruchomienie

```bash
npm run dev
```

Aplikacja będzie dostępna na **http://localhost:5173**

## Build

```bash
npm run build
```

## Preview build

```bash
npm run preview
```

## Struktura projektu

```
social-app-frontend/
├── src/
│   ├── components/    # Komponenty React
│   ├── services/      # API service
│   ├── App.jsx        # Główny komponent
│   └── main.jsx       # Entry point
├── public/            # Statyczne pliki
└── index.html         # HTML template
```

## Funkcjonalności

- Wyświetlanie listy postów
- Wyświetlanie listy użytkowników
- Polubianie postów
- Wyświetlanie i dodawanie komentarzy
- Usuwanie postów (tylko właściciel)
